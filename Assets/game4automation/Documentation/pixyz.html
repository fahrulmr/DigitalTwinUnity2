Import CAD Data based on PIXY

PIXYZ is a commercial add on solution for Unity which can import almost all CAD formats https://www.pixyz-software.com/. Game4Automation Professional is including an advanced PIXYZ import and update mechanism.

You need to use the latest PIXYZ 2018 Version (PiXYZ4Unity-2018.3.2.r3) together with Game4Automation. PIXYZ is not included in Game4Automation, you need to purchase it directly from PIXYZ.PIXYZ is providing a 7-day trial license.

We published a video tutorial which explains the update process based on PIXYZ: https://youtu.be/1IC12iZzL0w

Pixyz Importer

The Pixyz Importer script lets you import CAD data into the current scene. It is important, that PIXYZ is installed in your project and that PIXYZ is enabled in the Game4Automation menu:

You can define PIXYZ Import settings and set them under Import Settings. For importing, select a File with Select CAD file and press Import. The newly imported data will be imported as children of the component where Pixyz Importer is attached to with the name Imported Name.

For a first import unselect Update Import. For a second update import please select Update Import (see section CADUpdater).
