Game4Automation Installation

Install Unity

First install Unity ,if it is not already on your computer. In some cases Unity is free for private use, for commercial use you need a license. Please check on https://store.unity.com/ which version you need and install that.

This tutorial explains the first steps with Unity and Game4Automation:

Please make sure you are using Unity version 2018.3.10f1 and above, if you have already have an installed Unity version!

Create a new project

Start Unity and create an empty project by selecting new. Choose a Location and define a Project name for your new project. Then Create project.

It will now take a while for Unity to create the whole project structure. After the installation of Unity is finished you will get an empty Unity model:

Make sure that .Net4.0 is enabled

Usually, this should be the standard, but it might happen, if you integrate Game4Automation into an old project,that this setting is wrong. Please check, if you are not sure.

Go to the Player Settings by selecting Build Settings > Player Settings and change under Configuration the Scripting Runtime Version to .NET4.x Equivalent (see picture).

Get Game4Automation

Please go to the Unity Asset Store and puchase Game4Automation in the version Lite or Professional as required.

Import Game4Automation

Now we are ready to import the Game4Automation Package. You can do that directly over the asset store or manually download the asset.

For installing it manually please select from the main menu Assets > Import Package > Custom Package..:

This will show a new window with all the Game4Automation package contents:

Please select All and Import.

Check if a new menu is available

After the installation of the Game4Automation package you should see the additional game4automation folder in the project structure:

Also you will see a new Menu game4automation in the main menu bar.

Set standard settings

This step is not necessary because after the Game4Automation package import, all standard settings are applied automatically. The settings that are applied automatically are explained below.

Even while Game4Automation will work with default settings, we strongly recommend allowing Game4Automation to adjust the default settings. To apply the settings manualy, select game4automation>Standard Settings from the main menu.



This will create:

  • standard layer naming (see Physics)
  • standard physic collision matrix (see Physics)
  • standard light settings

At this stage everything is installed and you are now ready to continue to examine the Demomodell, that is outlined in section Demomodel.
