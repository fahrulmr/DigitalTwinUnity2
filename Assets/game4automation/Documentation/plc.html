PLC (ControlLogic)



If no real PLC is connected to the model, a PLC script (based on the ControlLogic base script) can be used to control the model. This can be used for replacing missing PLC functions during the development phase or for demo purposes. The Game4Automation Demo Model is fully controlled by PLC scripts.

This is a simple PLC-script from the Demo Model. This script is controlling the conveyor with the cans. As soon as the sensor at the end of the conveyor is high the conveyor is stopped.
